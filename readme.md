## Installing

Node:
```
apt-get install nodejs
apt-get install npm
```

MySql:
```
apt-get install mysql-server
If you have error "Cannot find module "mysql" -> npm install mysql
```

Apache2:
```
apt-get install apache2
```

PhpMyAdmin:
```
apt-get install phpmyadmin
(remember check apache2)
```
